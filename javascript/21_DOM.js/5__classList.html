ðŸ”¹ What is classList?

classList is a property that lets you add, remove, toggle, or check CSS classes on HTML elements easily.

ðŸ”¹ Syntax & Examples

// Select element
let box = document.getElementById("box");

// Add a class
box.classList.add("highlight");

// Remove a class
box.classList.remove("highlight");

// Toggle a class
box.classList.toggle("dark");

// Check if a class exists
box.classList.contains("highlight"); // returns true or false

// Replace a class
box.classList.replace("highlight", "dark");
