Manipulating Attributes in JavaScript

1. getAttribute(attributeName)

Used to get the value of an attribute.

Example:

let img = document.getElementById("myImage");
let srcValue = img.getAttribute("src");  // "image.jpg"
console.log(srcValue);

2. setAttribute(attributeName, value)

Used to set or update the value of an attribute.

img.setAttribute("alt", "Updated Image");  // changes alt text
img.setAttribute("width", "300");          // adds width attribute

3. removeAttribute(attributeName)

Used to remove an attribute from an element.

img.removeAttribute("width");

4. hasAttribute(attributeName)

Checks if an element has a specific attribute.

if (img.hasAttribute("alt")) {
  console.log("Alt attribute is present.");
}

Complete Example with Button

let btn = document.getElementById("btn");

console.log(btn.getAttribute("title"));        // "Click me"

btn.setAttribute("title", "Updated Title");

btn.removeAttribute("title");

console.log(btn.hasAttribute("title"));        // false
