Manipulating Style in JavaScript

Box

1. Inline Style Manipulation

You can change the style of an element using element.style.property.

let box = document.getElementById("box");
box.style.backgroundColor = "coral";
box.style.color = "white";
box.style.border = "2px solid black";

2. Accessing Computed Style

Use getComputedStyle() to get the final style of an element after all CSS is applied.

let computed = getComputedStyle(box);
console.log(computed.backgroundColor);

3. Toggle Styles Dynamically

You can also toggle styles using JavaScript to make elements interactive.

if (box.style.backgroundColor === "coral") {
  box.style.backgroundColor = "lightblue";
} else {
  box.style.backgroundColor = "coral";
}
