1. onclick

When it fires: When the user clicks on an element (e.g., button, div, etc).

Syntax: <button onclick="functionName()">Click</button>

<button onclick="sayHello()">Click Me</button>

<script>
  function sayHello() {
    alert("Hello from onclick!");
  }
</script>

2. onmouseenter

When it fires: When the mouse enters the boundary of an element.

Syntax: <div onmouseenter="doSomething()"></div>

<div onmouseenter="changeColor()">Hover over me</div>

<script>
  function changeColor() {
    alert("Mouse entered!");
  }
</script>

3. onmouseleave

When it fires: When the mouse leaves the element's boundary.

Syntax: <div onmouseleave="reset()"></div>

<div onmouseleave="resetColor()">Leave me</div>

<script>
  function resetColor() {
    alert("Mouse left!");
  }
</script>

4. onchange

When it fires: When the value of a form element changes (like <select>, <input type="checkbox">).

Syntax: <select onchange="showFruit(this)">

<select onchange="showFruit(this)">
  <option value="Apple">Apple</option>
  <option value="Banana">Banana</option>
</select>

<script>
  function showFruit(selectBox) {
    alert("Selected: " + selectBox.value);
  }
</script>

5. oninput

When it fires: When the user types or inputs data into a text field (fires immediately as the value changes).

Syntax: <input oninput="showInput(this)">

<input type="text" oninput="showInput(this)">

<script>
  function showInput(inputField) {
    console.log("You typed: " + inputField.value);
  }
</script>
